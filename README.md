# 📊 Pandas Practice Projects

## 📌 Project Overview
This project demonstrates basic data analysis using **Python and Pandas** on a sample dataset. The goal is to practice data cleaning, exploration, and extracting insights.

---

## 🎯 Objective
- Perform data analysis using Pandas  
- Practice filtering, sorting, and grouping  
- Generate insights from structured data  

---

## 🗂️ Dataset
A sample dataset of students with:
- Name  
- Marks  
- Gender  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- Jupyter Notebook  

---

## 🔧 Data Processing Steps
- Created DataFrame from dictionary  
- Checked structure using `df.info()`  
- Analyzed data using filtering and sorting  
- Applied basic aggregations  

---

## 📊 Key Analysis Performed
- Highest marks scored  
- Average marks  
- Gender-wise analysis  
- Top performing students  

---

## 📈 Sample Insights
- Highest marks scored: **99**  
- Average marks: *92.14*  
- Female students scored higher on average *(92.5)* than male students *(91.67)*  

---

## 📁 File in Repository
- `project.ipynb` → Main analysis notebook  
 

---

## 🚀 Conclusion
This project helped in understanding basic Pandas operations such as DataFrame creation, filtering, and aggregation. It serves as a foundation for more advanced data analysis tasks.
